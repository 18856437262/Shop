校园小菜

基于Bmob移动后端的资讯平台--HBUT版(Android)

Download this project as a .zip file Download this project as a tar.gz file

校园小菜，是一款基于Android的移动App, 可供运作于各大高校的在线生活咨询平台。其中提供了各种版块可用来整合高校各类资源， 如校园咨询、兼职招聘、宾馆KTV、餐馆食堂优惠、宿舍小超市、学习考研分享、租车旅游、景点门票等。只需要拥有一个强大的运营团队，就可以将校园的各大商家、学生聚合在一个平台上。一旦平台在本校区运作良好，通过校园小菜建立起一个比较全面的高校生活圈，便可成为校内强大的媒体，学生对外的接口，这个时候收益潜能不可估量。同时校园小菜可以快速在若干学校迭代，形成蝴蝶效应，也许那天强大的校园小菜高校联盟从此到来了。
